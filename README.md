# LandCoin
A Land Claiming system minecraft plugin using global Coin API.


The best free Land claiming plugin.

This plugin is a Chunk Based land claim plugin to protect your world.

It uses a global Coin currency to manage the plugin economy.

Depencency:

Vault - https://www.zrips.net/cmivault/

CoinCard - https://www.spigotmc.org/resources/coin-card.129770/

API - Website to login or register your Coin account. https://bank.foxsrv.net/

Discord Server - https://discord.gg/4nzsRJ7Ccq

Discord Bot - https://discord.com/oauth2/authorize?client_id=1391067775077978214

Chrome Extension (if you want 0-0) https://chromewebstore.google.com/detail/coin-bank/lbojdaalcfajcjphlpphjchkenjpbemf

Copy your coin card and do /coin card YOUR_CARD_ID (paste your id instead of your_card_id)

Now you can use the Coin economy globally!

And this is the plugin commands:

/land

/land claim

/land unclaim

/land area claim <name>

/land area unclaim

/land set <role> <break/place/access/use> <true/false>

/land rent

/land area rent

/land unrent [all/area]

/land rental <price>

/land area rental <price>

/land sell <price>

/land buy

/land info

/land view

/land trust <player> [role]

/land area trust <jogador> [role]

/land untrust <player>

/land area untrust <player>

/land reload

/selection

/selection clear

/selection info

/selection view

/land admin setowner <player>

/land admin unclaim

/land admin set <role> <break/place/access/use> <true/false>

/land admin selection

/land admin nextday
